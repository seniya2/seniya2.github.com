Data Storage / DBeaver

2015.10.27 : Data Storage 종류, DBeaver 툴 이용법

- 본 문서는 데이터스토로지 일반 및 DBeaver 프로그램을 설명한다.

1. Data Storage 종류

1-1) RDBMS

- 일반 DB : SQL Server(MSsql), PostgreSQL, MySQL, Oracle

- Memory DB : HSQL, Derby, H2

1-2) NoSQL

- Column Family Data Store : HBase, Cassandra

- Document Store : MongoDB

- Key/Value Store : Redis

 

 

2. DBeaver 이용법

2-1) 사이트 위치 : http://dbeaver.jkiss.org

2-2) 특징 : No need of installation. Its portable. Also this client supports more than 20 databases.

- 각 해당 데이터베이스 드라이브를 다운로드 받을 경우 인터넷 연결이 필요할 수 있다.
